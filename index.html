yes man sky Demo

Features :

  • Three.js terrain generation with a Perlin Noise and PlaneBufferGeometry.
  • The terrain is tiled. A big heightMap is generated with Perlin Noise, and depending on player position, we add the right tiles to the scene (demonstrating video and an image).
  • The height map is stored in a file using HTML5 FileSystem API.
  • Shader peregrinations : vertex (aka physical object) and fragment (aka colors)
  • Stars : displaying some stars, randomly, in a cone shaped centered at the camera, moving with it. At first I used a particle system, then I went with regular sprites - tried both, same perfs.
  • Water : a less static water ; still a wip, but the effect is not that bad. Using a texture, some random() for the waves, and 2 Planes to get a not so bad looking effect (video)
  • Fiddled with controls ... nothing satisfying yet.
